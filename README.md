# Mobil Bankacılık Uygulaması

Bu proje, Python ve Tkinter kullanılarak geliştirilmiş bir masaüstü bankacılık uygulamasıdır. Mobil arayüz boyutlarına uygun olacak şekilde tasarlanmıştır.

## Özellikler
- Hesap Açma ve Giriş Yapma
- Bakiye Sorgulama
- Para Yatırma ve Para Çekme
- Farklı Hesaplara Para Gönderme
- Kredi Çekme İşlemleri

## Kurulum ve Çalıştırma
Bu uygulama sadece standart Python kütüphanelerini kullanmaktadır. Herhangi bir dış bağımlılığa ihtiyacınız yoktur.

Projeyi çalıştırmak için:
```bash
python main.py
```
